1. Import and build the maven project in your favourite IDE.
2. Run DemoPayApplication.java as a java application. Or, use following maven command to run the same: mvn spring-boot:run
3. Access following rest URL:
http://localhost:8080/demo?sms=TOTAL-SENT-MSMITH
4. This will return the response by the SMS Handler.
5. Update the sms query parameter value to execute different command.
6. Supported commands are as mentioned in the initial specification document/problem statement provided.
